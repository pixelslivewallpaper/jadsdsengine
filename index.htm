JADSDS engine is a friendly tag based JavaScript animator created to make easier web projects using the power of HTML5












All the animations are built using intuitive tags easy to remember.
 

<property>

  <propertyName>Rotation</propertyName>

  <minimumDegreesValue>0</minimumDegreesValue>

  <maximumDegreesValue>360</maximumDegreesValue>

  <updateDegreesValue>6</updateDegreesValue>

  <currentDegreesValue>0</currentDegreesValue>

  <rotationSpeed>1</rotationSpeed>

  <animationRotationAction>Increase</animationRotationAction>

</property>

 


You do not need to be a programmer, just the basics about html and JavaScript will allow you to create amazing animations.
 
 

var jadsdsEngine = new JadsdsEngine('canvasID');

jadsdsEngine.loadAnimation('animationName');

Run multiple projects at the same time without cause any heavy impact on your CPU.
 
 


All your web projects run on Android as well.
 
 


Rotation, Scale, Video, Color cycling, Animated GIFs support, Move animation, Path animation, Audio, Alpha compositing, Color palettes, Endless animation, Events & Actions and much more. Jadsds has more than 100 tags available.
 












Play with the values, press “Run xml” and see what happens. If something goes wrong, just press “Reset”.

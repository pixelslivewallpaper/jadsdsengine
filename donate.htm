JADSDS engine is a friendly tag based JavaScript animator created to make easier web projects using the power of HTML5



- All the animations are built using intuitive tags easy to remember.

- You do not need be a programmer, just the basics about html and JavaScript will allow you to create amazing animations.

- Run multiple projects at the same time without cause any heavy impact in your CPU.

- All your web projects run in Android as well.
